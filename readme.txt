This plugin add custom widgets to the dashboard.  Widgets are customizable via settings/custom dashboard widgets

== Description ==

This plugin add two custom widgets to the dashboard.  Both Widgets are customizable via settings/custom dashboard widgets and they will always appear together on the top of the two columns inside the dashboard.  Also there's an option for selecting which user roles should see the widgets (Administrator, Editor, Author, Follower)

This plugin does the following:

*   Adds a new menu option under /settings/
*   Allows user to write custom Title and Content for widgets that are going to display only on the dashboard.  TODO: Display widgets or not depending on the user role.

== Installation ==

1. Upload the plugin PHP folder and file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. No other configuration needed
